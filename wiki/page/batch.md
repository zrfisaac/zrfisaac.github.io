# 💻 Batch

> *Última atualização: 2025-08-05*

Batch é um tipo de arquivo de script usado principalmente em sistemas Windows para automatizar tarefas repetitivas através da execução sequencial de comandos no prompt de comando (cmd). Esses arquivos com extensão `.bat` ou `.cmd` permitem criar rotinas simples para gerenciamento de arquivos, configuração do sistema e execução de programas, facilitando a automação sem a necessidade de software adicional.

## 💡 Exemplos

- **Loop de Arquivos**

```batch
for %%z in (*.txt) do (
  echo %%z
)
```
