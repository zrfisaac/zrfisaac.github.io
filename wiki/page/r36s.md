# 🎮 R36S

> *Última atualização: 2025-07-27*

O **R36S** é um console portátil compacto e versátil, conhecido por sua capacidade de rodar múltiplos emuladores de videogames clássicos, oferecendo suporte a diversas plataformas como SNES, Sega, GBA, entre outras. Com uma interface simples e conectividade para expansão de memória via cartões SD, o R36S é bastante apreciado por entusiastas de retro gaming que buscam uma experiência prática e acessível para jogar títulos antigos em qualquer lugar.

## [⚙️ Especificações](#📚-índice)

* `ARQUITETURA` : suporta arquiteturas ARM 32-bit e 64-bit.
* `BAT` : nível atual da **bateria**.
* `BATERIA` : bateria de 3200 mAh com duração aproximada de 6 horas.
* `BRT` : **brilho** da tela.
* `PLACA` : placa dedicada R36S, não Raspberry Pi, com SoC Rockchip RK3326.
* `PROCESSADOR` : RK3326 Quad-core ARM 64-bit Cortex-A35 CPU até 1,5 GHz.
* `RAM` : 1 GB DDR3L.
* `SND` : volume de **som**.
* `TELA` : tela LCD de 3,5 polegadas (resolução 640x480 pixels).
* `WIFI` : estado do **Wi-Fi** (ligado/desligado).

## [⚙️ Comandos Úteis](#📚-índice)

* 🖥️ Faz backup dos primeiros 446 bytes do disco (geralmente o MBR).

```bash
sudo dd if=/dev/sdX of=r36s-00.img bs=1 count=446
````

* 💾 Restaura os primeiros 446 bytes do disco a partir do arquivo de imagem.

```bash
sudo dd if=r36s-00.img of=/dev/sdX bs=1 count=446
```

* 🅱️ Formata a partição como FAT16 e dá o nome "BOOT".

```bash
sudo mkfs.vfat -F 16 -n BOOT /dev/sdX1
```

* 🆁 Formata a partição como ext4 com label "root".

```bash
sudo mkfs.ext4 -L root /dev/sdX2
```

* 🅴 Formata a partição como exFAT com o nome "EASYROMS".

```bash
sudo mkfs.exfat -n EASYROMS /dev/sdX3
```

* 🔄 Copia arquivos recursivamente, ignorando já existentes, com limite de banda de 2000 KB/s e mostrando progresso.

```bash
rsync -av --ignore-existing --no-whole-file --partial --progress --bwlimit=2000 /source/ /destination/
```

* 💽 Grava um arquivo `.iso` diretamente no disco com limite de velocidade de 2 MB/s e mostra o progresso.

```bash
sudo pv -L 2m r36s.iso > /dev/sda
```

## [💾 ROMs Suportadas](#📚-índice)

### [🎮 **MS-DOS**](#📚-índice)
Jogos e aplicativos para o sistema MS-DOS, rodando via emuladores específicos, permitindo acesso a clássicos do PC dos anos 80 e 90.

### [🎮 **SNES (Super Nintendo Entertainment System)**](#📚-índice)
ROMs de jogos para o console 16-bit da Nintendo, famosos por seus títulos icônicos como *Super Mario World* e *The Legend of Zelda: A Link to the Past*.

### [🎮 **Sega 32X**](#📚-índice)
Extensão para o Sega Genesis/Mega Drive que oferecia gráficos melhorados e jogos exclusivos. O sistema suporta ROMs para essa plataforma, permitindo acesso a títulos menos comuns.

### [🎮 **PICO-8**](#📚-índice)
Um "fantasy console" para jogos retro criados com estilo pixel-art e limitação técnica intencional. Ideal para jogos indie e experimentais criados nessa plataforma.

### [🎮 **GBA (Game Boy Advance)**](#📚-índice)
Consola portátil da Nintendo que trouxe gráficos avançados para seu tempo. O sistema suporta jogos de GBA para quem quer jogar clássicos portáteis.

### [🎮 **GBC (Game Boy Color)**](#📚-índice)
Consola portátil anterior ao GBA, com suporte a cores e uma biblioteca extensa de jogos. Compatível para jogadores que preferem essa geração.

## [🐞 Possíveis Bugs](#📚-índice)

### [🔇 Áudio para de funcionar](#📚-índice)

Em alguns casos, o áudio pode parar de funcionar sem motivo aparente.  
Uma solução simples é **inserir um fone de ouvido na saída P2** e depois **removê-lo** — isso costuma fazer o som voltar ao normal.

### [⚠️ Problemas ao gravar imagem no cartão SD](#📚-índice)

Ao tentar gravar uma imagem diretamente no cartão SD usando um **adaptador conectado à porta USB 3.0**, o processo pode falhar ou travar.  
Para evitar isso, **utilize uma porta USB 2.0** durante a gravação da imagem.  
> **Observação:** se for apenas copiar arquivos comuns para o cartão, a porta USB 3.0 geralmente funciona sem problemas.

### 🧩 Dispositivo pode ser um clone

Alguns modelos adquiridos, especialmente de fontes não oficiais, podem ser **clones do K36** em vez do R36S original.  
Nesses casos, apenas o **ArkOS do K36** funciona corretamente — outras versões podem não inicializar ou apresentar falhas.
