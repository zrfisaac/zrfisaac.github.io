# 🧩 x86

O x86 é uma arquitetura de CPU de 16, 32 e 64 bits, criada pela Intel em 1978 com o Intel 8086. É uma das arquiteturas mais populares do mundo, usada em PCs, laptops e servidores.

> *Última atualização: 2025-08-13*

## 🗄️ Registrador

| Registrador | Tamanho | Função / Observação                                 |
| ----------- | ------- | --------------------------------------------------- |
| `EAX`       | 32 bits | Registrador acumulador (aritmética, I/O)            |
| `EBX`       | 32 bits | Base, usado em operações de endereço ou dados       |
| `ECX`       | 32 bits | Contador, usado em loops e shift/rotate             |
| `EDX`       | 32 bits | Dados, também usado em multiplicação/divisão        |
| `ESI`       | 32 bits | Source index (para operações com strings)           |
| `EDI`       | 32 bits | Destination index (strings)                         |
| `EBP`       | 32 bits | Base pointer, referência da pilha de função         |
| `ESP`       | 32 bits | Stack pointer, topo da pilha                        |
| `EIP`       | 32 bits | Instruction pointer (endereço da próxima instrução) |
| `EFLAGS`    | 32 bits | Flags de condição: CF, ZF, SF, OF, etc.             |

> Registradores de 16 bits (lower half): `AX, BX, CX, DX, SI, DI, BP, SP`
> Registradores de 8 bits (lower e higher): `AL/AH, BL/BH, CL/CH, DL/DH`

## 📝 Instrução

| Instrução   | Descrição                                    |
| ----------- | -------------------------------------------- |
| `ADD`       | Soma `src` em `dest`                         |
| `AND`       | AND lógico entre `dest` e `src`              |
| `CALL`      | Chama função (salva endereço de retorno)     |
| `CMP`       | Compara `op1` com `op2` (altera flags)       |
| `DEC`       | Decrementa `reg/mem` em 1                    |
| `IDIV`      | Divisão com sinal                            |
| `IMUL`      | Multiplicação com sinal                      |
| `INC`       | Incrementa `reg/mem` em 1                    |
| `JMP`       | Salto incondicional                          |
| `JE / JZ`   | Salta se igual / zero                        |
| `JNE / JNZ` | Salta se diferente / não zero                |
| `JG / JNLE` | Salta se maior (signed)                      |
| `JL / JNGE` | Salta se menor (signed)                      |
| `JGE / JNL` | Salta se maior ou igual (signed)             |
| `JLE / JNG` | Salta se menor ou igual (signed)             |
| `LEA`       | Carrega endereço de memória em registrador   |
| `MOV`       | Move dados de `src` para `dest`              |
| `MUL`       | Multiplicação sem sinal                      |
| `NEG`       | Inverte sinal de `reg/mem`                   |
| `NOT`       | Inverte todos os bits de `reg/mem`           |
| `OR`        | OR lógico entre `dest` e `src`               |
| `PUSH`      | Coloca valor na pilha                        |
| `POP`       | Remove valor da pilha para `reg/mem`         |
| `RET`       | Retorna de função                            |
| `SUB`       | Subtrai `src` de `dest`                      |
| `TEST`      | AND lógico entre `op1` e `op2`, altera flags |
| `XOR`       | XOR lógico entre `dest` e `src`              |
