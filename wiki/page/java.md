# ☕ Java

> *Última atualização: 2025-07-24*

Java é uma linguagem de programação de propósito geral, orientada a objetos e amplamente utilizada para desenvolvimento de aplicações que vão desde softwares para dispositivos móveis até sistemas corporativos complexos. Criada na década de 1990 pela Sun Microsystems, Java é conhecida por seu princípio “write once, run anywhere” (escreva uma vez, execute em qualquer lugar), graças à máquina virtual Java (JVM) que permite executar o mesmo código em diferentes plataformas sem modificações. Além disso, Java possui uma vasta biblioteca padrão, forte comunidade e é muito usada em aplicações web, Android, sistemas embarcados e enterprise.
