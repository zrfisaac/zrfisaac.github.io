# 🏛️ Delphi

> *Última atualização: 2025-07-24*

Delphi é uma linguagem de programação e ambiente de desenvolvimento integrado (IDE) baseado em Object Pascal, criado pela Borland. É amplamente usado para desenvolver aplicações desktop, móveis e sistemas embarcados com alta performance, oferecendo uma interface visual para criação rápida de interfaces gráficas e acesso facilitado a bancos de dados, além de suporte a múltiplas plataformas.
