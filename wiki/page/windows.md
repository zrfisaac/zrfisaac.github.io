# <img src="/wiki/icon/windows.png" alt="Ícone" width="24"> Windows

> *Última atualização: 2025-08-11*

O Windows é o sistema operacional da Microsoft que domina o mercado de computadores pessoais desde seu lançamento em 1985. Ele é conhecido por sua interface gráfica de usuário (GUI), que utiliza o conceito de "janelas" para exibir programas e arquivos, tornando a interação com o computador muito mais intuitiva do que os sistemas baseados em linha de comando que existiam na época, como o MS-DOS.

- 📂 Caminhos
  - 📂 `Inicialização:` C:\Users\%USERNAME%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
  - 📂 `Inicialização:` C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup
  - 📂 `Menu Iniciar:` C:\ProgramData\Microsoft\Windows\Start Menu
  - 📂 `Menu Iniciar:` C:\Users\\%USERNAME%\AppData\Roaming\Microsoft\Windows\Start Menu
  - 📂 `Usuário:` C:\Users\%USERNAME%
