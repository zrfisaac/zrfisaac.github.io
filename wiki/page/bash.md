# 💻 Bash

> *Última atualização: 2025-07-24*

Bash (Bourne Again SHell) é um interpretador de comandos e linguagem de script amplamente usado em sistemas Unix e Linux. Ele permite que usuários e scripts executem comandos, automatizem tarefas e controlem o sistema operacional por meio de uma interface de linha de comando poderosa e flexível.
